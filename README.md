# Timetable_Generater-ClassIQ
# 🗓️ Timetable Generator System

A web-based **Timetable Generator Project** designed for Computer Engineering colleges to automatically generate and manage class timetables efficiently.

---

## 🚀 Features

- Automatic timetable generation
- Teacher-wise scheduling
- Classroom management
- Lab batch scheduling support
- Avoids time clashes (teacher & room conflict check)
- Easy-to-use web interface
- Save and view generated timetables

---

## 🛠️ Technologies Used

- HTML
- CSS
- JavaScript
- Node.js (if backend used)
- Express.js
- MongoDB (for database)
